# hospital management system for toamedit

Current software version: v2.0.0
 
Face recognition models (for face attendance testing)
- The app expects face-api.js models under `public/models` for the registration page.
- To download models automatically on Windows, run:

```powershell
.\scripts\download-face-models.ps1
```

If the script fails, download the models manually from a face-api.js models repository and place them in `public/models`.
